# 0.2.11 (2026-04-20)

這個版本把前一輪收斂到一半的閱讀器與書源功能補成可交付狀態。主軸很明確：閱讀器互動收口、書源驗證閉環補齊、離線快取語意對齊，以及把閱讀紀錄與設定頁清乾淨。

## 亮點

- 閱讀器的翻頁模式、日夜主題與自動翻頁行為全部重新收口，切換後會立即生效
- 書源驗證現在有完整閉環，支援瀏覽器驗證與驗證碼輸入，不再只停在背景 service
- 書籍詳情頁的下載入口正式改成 `離線快取`，語意與實際行為一致
- 閱讀紀錄時間單位修正，About / 設定頁只保留一套正式入口

## 詳細更新

### 閱讀器

- 修正翻頁模式切換後要退出重進才會生效的問題
- 閱讀控制選單改成獨立樣式，不再直接和閱讀背景混在一起
- 日夜主題切換會在保存的白天 / 夜間主題之間切換，並用 `charOffset` 保住閱讀位置
- 自動翻頁預設改為更合理的速度，且會在手動翻頁、切模式、切主題時正確重置或恢復進度
- 重新規劃頂部與底部閱讀留白，避免正文蓋到書名 / 百分比列
- 離開再回到書本時，restore / repaginate 對齊邏輯更穩定

### 書源 / 驗證 / 快取

- 新增書源驗證協調層，驗證請求現在能從 service 正確送到 UI 並回傳結果
- 支援瀏覽器驗證頁與驗證碼輸入對話框
- 補強 timeout / cancel / dispose 競態處理，減少校驗時的閃退與 pending 卡死
- 書源閱讀抓到的章節內容現在會持久化到本地快取
- 書籍詳情頁可直接建立離線快取任務，支援整本、從目前進度開始與只補未快取章節

### 設定 / 紀錄 / 頁面清理

- TTS 已可列出系統 engine / voice，並保存語音選擇
- 閱讀紀錄修正為以秒數正確顯示時長
- About 與設定頁統一使用正式的閱讀紀錄頁，移除重複實作
- 清掉閱讀設定、其他設定與「我的」頁中一批尚未實作的入口
- 發現頁搜尋收斂成頂部單一搜尋列，分類改成四欄排列
