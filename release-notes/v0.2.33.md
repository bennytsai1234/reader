# 0.2.33 (2026-04-27)

這個版本集中重構閱讀器核心，將閱讀顯示改成 tile-based 視窗模型，優先解決正文重複切行、捲動掉幀與平移翻頁空白閃爍。

## 閱讀器核心

- 重寫 `LayoutEngine` 的分行邏輯，修正中文標點與視覺行邊界造成的重複行問題
- 新增 `ReaderTilePainter`、`ReaderTileLayer` 與 `TileKey`，把文字繪製拆成可重用的單 tile layer
- 重寫 scroll 模式 viewport，改為 `Stack + Transform.translate` 移動 tile，避免拖動時每幀重畫整頁文字
- 重寫 slide 模式 viewport，改成先 recenter 再 commit window，修正翻頁過渡中的空白與跳頁
- 補上 tile 導向的 runtime API 與前後預載流程，統一兩種閱讀模式的移動與邊界策略

## 測試

- 新增 `LayoutEngine` 回歸測試，覆蓋中文標點、長標題、空章節與 offset 單調性
- 新增 tile viewport widget tests，驗證 scroll 只更新 transform、slide 先 recenter 再 commit
- 保留並通過既有 reader runtime / layout / placeholder 邊界測試
