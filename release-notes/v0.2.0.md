# 0.2.0 (2026-04-10)

## 🚀 主要重構與新功能 (Major Refactoring & Features)

* **搜尋架構重構 (對標 Legado)**:
  * 徹底重構了搜尋系統架構，將原本的 God Object (SearchProvider) 解耦為更清晰的三層架構 `SearchProvider (UI State)` -> `SearchModel (Engine)`。
  * 實作全新的 `SearchScope` 機制，支援「全部書源」、「分類書源」與「單一書源」的細粒度搜尋。
  * 新增 `SearchScopeSheet` UI 元件，允許使用者在搜尋時直觀地選擇不同書源或群組。
  * 優化搜尋歷史，增強了長期持久化儲存與長按刪除單個歷史詞條功能。
* **發現頁面重構 (Explore 升級)**:
  * 將原本的發現頁面 (Explore) 的基礎架構完全顛覆，對齊 Legado 雙層書源架構。
  * 移除舊版不合時宜的年齡驗證 (`is18Plus`) 判斷。
  * 改以基於書源的分類設計進行檢索，實現更靈活的書單與排行展現模式。
* **書源管理最佳化 (Source Manager)**:
  * 書源管理模組介面與功能全面優化更新。

## 🐛 修復與效能優化 (Fixes & Optimizations)

* 修復了切換章節（Slide機制）時，如果沒有正確延遲重置動畫而發生的畫面跳動、閃開問題。
* 修正滑動換頁進度在模式切換後未正確重置的異常。
* 消除了多個廢棄元件，移除了 `SearchService` 與多個不再使用的代碼段，精簡專案體積。
* 修復了所有遺留的 `flutter analyze` 語法過時 (deprecation warnings) 與分析錯誤。
