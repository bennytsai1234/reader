# 0.2.26 (2026-04-25)

這個版本補齊 `0.2.25` 後仍可能出現的 scroll mode 初始還原卡死問題。修正重點是統一 scroll restore 的 15% anchor 座標系，避免目標位置已經到達但 navigation confirmation 永遠無法完成。

## 修正

- scroll restore 的 anchor padding 改用實際 scroll viewport 高度計算，納入 top/bottom inset
- 章節內容 ready 後的 restore retarget 會以同一套 anchor padding 對齊，不再混用原始 `localOffset`
- pending scroll restore 會優先於 stale chapter jump 執行，並清掉舊的 pending jump
- restore transaction 新增 visible location match 保底，避免因 anchor 像素誤差永久鎖住 scroll input
