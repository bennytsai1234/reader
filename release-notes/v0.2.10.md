# 0.2.10 (2026-04-20)

這個版本是一次小型穩定版發佈，主軸很單純：把 TTS 設定同步補齊、把 analyzer 清回全綠，並把 repo 內一批已過時的文檔改寫成以目前程式碼為準的版本。

## 亮點

- `朗讀與語音` 頁面的語速 / 音調現在會即時作用到目前的 `TTSService`
- TTS 預設語速與音調統一為 `1.0`
- 閱讀器返回行為改用 `PopScope`，對齊新版 Flutter
- `flutter analyze` 回到 `No issues found!`

## 詳細更新

### TTS

- 修正設定頁只保存語速 / 音調、但不即時同步目前朗讀服務的問題
- 啟動後載入設定時，會把語速 / 音調 / 音量重新灌回全域 `TTSService`
- 閱讀器內部的 TTS fallback 預設也統一為 `1.0`

### 閱讀器 / 互動

- 將 `WillPopScope` 換成 `PopScope`，避免新版 Flutter 下 predictive back 提示
- 順手整理返回流程相關的 async context 用法，減少 analyzer 噪音

### 文檔 / 工程整理

- 重寫 `docs/architecture.md`、`docs/reader_architecture_current.md`、`docs/DATABASE.md`
- 重寫 `docs/roadmap.md`、`docs/next_stage_handoff.md`、`docs/slide-chapter-preload-plan.md`
- 將 `docs/source_audit_backlog.md` 改成 `reader` 與 `legado` 的手動對照稿
- 清理剩餘 lint / warning，讓 analyzer 恢復全綠
